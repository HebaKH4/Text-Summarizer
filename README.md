# AI Text Summarizer

A hybrid NLP-based text summarization web tool that combines:

- Extractive summarization using TF-IDF
- Abstractive summarization using BART (Hugging Face Transformers)

## Features
- Summarizes long text
- Web interface (HTML/CSS/JS)
- Flask backend API

## Tech Stack
- Python
- Flask
- Transformers (BART)
- Scikit-learn
- JavaScript / HTML / CSS

## How to Run

1. Install dependencies:

pip install -r backend/requirements.txt

2. Run backend:

python backend/app.py

3. Open index.html in browser
